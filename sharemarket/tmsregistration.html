Things needed for TMS User Registration

  1. DEMAT Number (BOID)

    DEMAT Number contains 16 digit number starting with 130

  2. Mobile Number
  3. Gmail/Email
  4. Bank Account Number
  5. Scan PP Size Photo

  6. Scan Signature
  7. Scan Thumbprints

  8. PAN Number (optional)

  9. Selfie carrying Citizenship

  10. Scan copy of Citizenship
